How to connect your Excel file

  1. Upload NBDHE_Subject_Breakdown.xlsx to Google Drive
  2. Right-click the file → Share → set to "Anyone with the link" → Viewer → Copy link
  3. Your link looks like: docs.google.com/spreadsheets/d/FILEID/edit
  4. Copy just the bold part between /d/ and /edit
  5. Open this HTML in a text editor, find the line:
    const DRIVE_FILE_ID = "YOUR_GOOGLE_DRIVE_FILE_ID_HERE";
  6. Replace YOUR_GOOGLE_DRIVE_FILE_ID_HERE with your file ID
  7. Save and re-upload this HTML to Google Sites
